ページ内で有効な変数を定義することができる。※積極的にやることではない。

{% set count = 1 %} {% set count = count + 1 %} {{ count }}

スコープ(有効範囲)を決めた変数の利用。

{% with i = 0 %} {{ i }} {% endwith %} {# ↓ これは出ない #} {{ i }}

余分な空白文字の除去。

{#- 手前空白除去 #}AAA
AAA{# 後方空白除去 -#}
AAA {#- 前後空白除去 -#} BBB
{{- 1 + 1 -}}
{%- if True -%} 123 {%- endif -%}
